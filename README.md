# Инструкция по запуску проекта

## Запуск проекта

1. Для запуска файла выполните команду:

   ```bash
   java -jar target/SimpleChatNewJava17-0.0.1-SNAPSHOT.jar
2. После этого откройте браузер и перейдите по адресу:
http://localhost:8080/api/v1/users .
3. Чтобы завершить работу проекта, в командной строке зажмите комбинацию клавиш Ctrl+C.

## Сборка проекта
Если файл SimpleChatNewJava17-0.0.1-SNAPSHOT.jar отсутствует в папке target, необходимо собрать проект: 


4. Для этого выполните одну из следующих команд:
   `mvnd clean package` или `mvn clean package`
## Установка Maven
Если команда mvnd или mvn не распознана в командной строке, вам нужно установить Maven:


5. Перейдите на официальный сайт Maven и скачайте последнюю версию (https://maven.apache.org/download.cgi).


6. Распакуйте архив Maven и укажите путь к его папке (например, C:\path\to\maven) в системной переменной Path.


## Добавление в переменную среды
После установки Maven слудует добавить её команды в переменную среды Path:


7. Перейдите в Свойства компьютера. 


8. Выберите Дополнительные параметры системы. 


9. Нажмите Переменные среды. 


10. В разделе Системные переменные найдите переменную Path и добавьте в нее путь к папке bin из распакованной директории Maven (например, C:\path\to\maven\bin). 


11. После этого перезапустите командную строку, и команды mvn и mvnd будут доступны для использования.


Используемая версия java - 17.0.12
