# Инструкция по запуску проекта

## Запуск проекта

1. Для запуска файла выполните команду:

   ```bash
   java -jar target/SimpleChatNewJava17-0.0.1-SNAPSHOT.jar
После этого откройте браузер и перейдите по адресу:
http://localhost:8080/api/v1/users .
Чтобы завершить работу проекта, в командной строке зажмите комбинацию клавиш Ctrl+C.

## Сборка проекта
Если файл SimpleChatNewJava17-0.0.1-SNAPSHOT.jar отсутствует в папке target, необходимо собрать проект. Для этого выполните одну из следующих команд:

`mvnd clean package`
или `mvn clean package`
## Установка Maven
Если команда mvnd или mvn не распознана в командной строке, вам нужно установить Maven:

Перейдите на официальный сайт Maven и скачайте последнюю версию.

Распакуйте архив Maven и укажите путь к его папке (например, C:\path\to\maven) в системной переменной Path.

Шаги для добавления в переменную среды:
Перейдите в Свойства компьютера.

Выберите Дополнительные параметры системы.

Нажмите Переменные среды.

В разделе Системные переменные найдите переменную Path и добавьте в нее путь к папке bin из распакованной директории Maven (например, C:\path\to\maven\bin).

После этого перезапустите командную строку, и команды mvn и mvnd будут доступны для использования.

Используемая версия java - 17.0.12
